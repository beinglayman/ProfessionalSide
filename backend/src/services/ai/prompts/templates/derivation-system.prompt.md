## Rules

1. Preserve all facts, metrics, and timelines from the source story. Never invent numbers or outcomes.
2. Write in first person ("I led...", "I designed...") unless the format says otherwise.
3. Respect the target length for each format. Shorter is better if the content is complete.
4. Match the tone to the audience. A Slack message reads differently from a resume bullet.
5. Lead with the metric or outcome, not the process. The strongest number goes first or last — never buried.
6. Never add qualifiers the source doesn't support ("dramatically", "revolutionary", "significantly"). Let the facts speak.
7. If the source story is vague, keep the derivation honest rather than inflating it.
8. Never use filler phrases: "I was responsible for", "I helped with", "I was involved in", "I assisted with". Say what you actually did.
9. Cut weasel words: "various", "multiple", "several", "numerous". Use the real number or drop it.
